# `Cartoon Characters`

## To setup the project

1) Clone this repo
```
git clone https://github.com/gresiPatel/cartoon_characters.git
```

2) Intall dependencies
```
npm install
```

3) To link dependencies into ios
```
cd ios
pod install
```

## To run the project

- **iOS:**
```
npx react-native run-ios
```

- **Android:**
```
npx react-native run-android
```

- **Web:**
```
npm run start:web
```
